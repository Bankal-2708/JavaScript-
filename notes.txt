+++++++++++++++++++++++++++++ REACT +++++++++++++++++++++++++++++

React is a JavaScript library that is use to build a user interface(UI), especially single page applications by using reusable components

it was created by meta/facebook

COMPONENTS : components are the reusable, independent piece of code of ui 

PROPS : it means properties are how pass data from parents to child components, when we pass data by props we cant modify this only for read and use (props are immutable)

STATE : state is data that belongs to a components and change over time, when the state change react automatically re-render the component and update the changes on UI 

USESTATE : it returns an array with two items, the current state and a function to update it

USEEFFECT : it is a react hook that let you perform side effects in a functional component without disturbing the main thread, it run you code after your component re-render(featching data from api, setTimout, setInterval etc...)

its dependencies : [] => run only once when page load
[count] => when count chnage
no dependencies => run on every re-rendering
