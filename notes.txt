+++++++++++++++++++++++++++++ REACT +++++++++++++++++++++++++++++




React is a JavaScript library that is use to build a user interface(UI), especially single page applications by using reusable components

it was created by meta/facebook

COMPONENTS : components are the reusable, independent piece of code of ui 

PROPS : it means properties are how pass data from parents to child components, when we pass data by props we cant modify this only for read and use (props are immutable)

STATE : state is data that belongs to a components and change over time, when the state change react automatically re-render the component and update the changes on UI 

USESTATE : it returns an array with two items, the current state and a function to update it

USEEFFECT : it is a react hook that let you perform side effects in a functional component without disturbing the main thread, it run you code after your component re-render(featching data from api, setTimout, setInterval etc...)

its dependencies : no dependencies => run on every re-rendering
[] => run only once when page load
[count] => when count chnage


DOM : document object model it's a tree like structure or representation of the html page that browser create it in memory

a dom is a tree like structure that connect every file or node or components of projects

if you changeanything in component then dom will update that change, [Everything re-render in project]

VIRTUAL DOM : virtual dom is a light weighted, in React/ computer memory , it is the copy made by the javascript of the real dom insted of updating the dom direclty, react update the virtual dom first 

* three steps of virtual dom 

1) New Blueprint : when something change react create a new virtual dom tree in background
2) Diffing Algo (Spotting the diff) :
  a) react compare the virtual dom with real dom 
  b) it find which branch or leaf of the tree change 
  c) it mark them

3) RECONCILLIATION : updae only those specific part of the real dom

direclty updating the real dom is SLOW (it repaint everything)
